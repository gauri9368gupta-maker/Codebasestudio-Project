# 📸 Advanced Image Gallery with Modal

An interactive and responsive Image Gallery built using **HTML, CSS, and JavaScript**. This project allows users to browse images, search, filter by categories, upload new images, mark favorites, manage trash, and view images in a fullscreen modal.

---

## ✨ Features

- 📂 Category Filtering
  - All Images
  - Nature
  - Travel
  - Animals
  - Cars

- ❤️ Favorites
  - Mark images as favorites.
  - Heart icon turns red when selected.
  - View all favorite images using the Favorites filter.

- 🗑️ Trash Management
  - Move images to Trash.
  - Restore deleted images.
  - Permanently delete images with confirmation.

- 🔍 Search Images
  - Search images instantly by title.

- 📤 Upload Images
  - Upload multiple images from your device.
  - Newly uploaded images are added to the gallery instantly.

- 🖼️ Image Modal
  - View images in a larger popup.
  - Previous and Next navigation.
  - Image counter.
  - Caption display.

- ⬇️ Download Images
  - Download any image directly from the modal.

- 🖥️ Fullscreen Mode
  - Open images in fullscreen.

- ⌨️ Keyboard Navigation
  - Right Arrow → Next Image
  - Left Arrow → Previous Image
  - Esc → Close Modal

- 📱 Touch Swipe Support
  - Swipe left/right to navigate images on touch devices.

- ❤️ Double Click Animation
  - Double-click an image in the modal for a heart animation.

- ⏳ Loading Screen
  - Smooth loader displayed while the gallery initializes.

- 📊 Dynamic Image Counter
  - Displays the number of currently visible images.

- 📱 Responsive Design
  - Works across desktop, tablet, and mobile devices.

## 🎮 How to Use

### Browse Images

Click on any image to open it in the modal viewer.

### Filter Images

Use the filter buttons to display images by category.

### Search

Type in the search bar to find images by title.

### Add Favorites

Click the ❤️ icon on any image to mark it as a favorite.

### Trash Images

Click the 🗑️ icon to move an image to the Trash.

### Restore Images

Open the Trash filter and click the Restore icon.

### Permanently Delete

Click the Delete icon inside Trash and confirm the action.

### Upload Images

Click the **Upload Images** button and select one or more images from your computer.
